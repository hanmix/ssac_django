# ssac_django
SeSAC 풀스택 웹 융합 개발자 양성 과정 3기 실습 및 개인 프로젝트

학습 내용 
 - NCP와 AWS로 나만의 서버를 구축
 - Django와 Node.js로 서버 환경 세팅 및 웹페이지 구현

Django 서버를 구축하기 위한 repository 였으나, Node.js 공부를 위해 해당 repository에 Node.js 프로젝트를 추가함.
